# File-sharing-online

A modern, easy-to-use web application for uploading and sharing files.

## Features

- 📤 **Drag & Drop Upload**: Simply drag files into the upload area or click to browse
- 📁 **Multiple File Support**: Upload multiple files at once
- 🎨 **Beautiful UI**: Clean and modern interface with gradient design
- 📥 **Download Files**: Download your uploaded files with one click
- 🔗 **Share Files**: Share files using the native share API or copy links
- 🗑️ **File Management**: Delete files you no longer need
- 📱 **Responsive Design**: Works on desktop, tablet, and mobile devices

## File Type Support

The application supports all file types and displays appropriate icons for:
- Images (🖼️)
- Videos (🎥)
- Audio files (🎵)
- PDFs (📄)
- Documents (📝)
- Spreadsheets (📊)
- Archives (🗜️)
- Text files (📃)
- And more!

## How to Use

1. Open `index.html` in your web browser
2. Click the "Choose Files" button or drag files into the upload area
3. Your files will appear in the "Uploaded Files" section
4. Use the action buttons to:
   - **Download**: Save the file to your device
   - **Share**: Share via native share or copy link
   - **Delete**: Remove the file from the list

## Installation

No installation required! Simply:

1. Clone this repository:
   ```bash
   git clone https://github.com/manavdev468/File-sharing-online.git
   ```

2. Open `index.html` in your web browser

## Technology Stack

- **HTML5**: Structure and file input handling
- **CSS3**: Modern styling with gradients and animations
- **JavaScript (Vanilla)**: File handling and interactivity
- **File API**: Browser-based file management

## Note

This is a client-side application. Files are stored temporarily in the browser's memory and are not uploaded to any server. When you refresh the page, the file list will be cleared.
